# 🐍 Snake Game

## 📌 Description
The Snake Game is a classic arcade-style browser game developed using HTML, CSS, and JavaScript. In this game, the player controls a snake that moves across a grid and collects food to grow longer. Each time the snake eats food, the score increases.

The game uses the HTML canvas element to render graphics dynamically. JavaScript handles the core functionality such as movement, collision detection, score tracking, and game logic. The player controls the snake using keyboard arrow keys.

The game ends when the snake collides with the wall or with itself. A real-time score display is shown, and the game can be restarted after a game-over.

---

## 🛠️ Technologies Used
- HTML (Canvas structure)
- CSS (Styling)
- JavaScript (Game logic)

---

## 🚀 Features
- Smooth snake movement  
- Random food generation  
- Score tracking  
- Collision detection  
- Game over functionality
